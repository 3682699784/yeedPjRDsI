# 项目简介

## 核心功能
- 登录与注册
- 用户密码修改
- 管理员与普通用户区分
- 图书信息管理
- 购物车功能
- 商品定价
- 商品审核
- 商品上架
- 商品推荐
- 商品查找
- 商品删除
- 商品修改
- 信息唯一性判断

## 适用场景
- 用于在线书店的搭建
- 适用于图书销售和管理的电商平台
- 适合作为个人或小型企业图书销售系统

## 技术特点
- 使用 Django 框架，简化开发流程
- 基于 MySQL 数据库，保障数据安全
- 前端采用 HTML、CSS 和 JavaScript，提供良好的用户体验
